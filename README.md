# minimaFetch

A tiny, dependency-free system fetch for Arch Linux. Shows a small ASCII
Arch logo next to your username, uptime, and root filesystem storage usage.

## Usage

```sh
git clone https://github.com/Joekrry/minimaFetch.git
```

### Run automatically on shell startup

To have minimaFetch run every time you open a new terminal, add it to your
shell's config file.
